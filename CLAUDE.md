# GRIDLOCK — 미로형 TD → 미니 RTS

## 명령어
npm run dev (개발 서버) / npm run build (배포 빌드)

## 문서
- 게임 규칙·수치 근거: DESIGN.md
- 작업 목록: PLAN.md — 한 세션에 한 태스크만 진행

## 아키텍처 규칙
- Vanilla TypeScript + Canvas 2D. 라이브러리 추가 금지 (제안은 가능, 사용자 승인 필요)
- update(dt)와 render(ctx) 분리. 렌더 함수에서 게임 상태 변경 금지
- 밸런스 수치는 전부 src/data/*.json. 코드에 매직넘버 금지
- 엔티티는 클래스 + 배열 관리. ECS 패턴 도입 금지
- 파일이 300줄 넘으면 분리를 제안할 것

## 작업 규칙
- PLAN.md의 태스크 단위로만 작업. 요청받지 않은 범위 확장 금지
- 태스크 완료 시: 수동 테스트 방법 안내 → 사용자 확인 후 커밋
- 대규모 리팩토링은 제안만 하고 승인 전 실행 금지
- 버그 수정 시 원인을 한 줄로 설명할 것
